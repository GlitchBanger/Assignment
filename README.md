To view my assignment 1) start the backend express app 2) run vue serve

1) Start a Terminal in the project root

$ cd ./todobackend
$ node index.js

2) Start another terminal in the project root

$ cd ./todoui
$ npm run serve

Make sure you have node and npm installed
